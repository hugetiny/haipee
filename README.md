## ✨ 核心特性

### 🤖 聚合顶尖 AI 模型
不再被单一厂商绑定。自由切换 **Gemini、Claude、OpenAI、DeepSeek** 等全球顶尖模型，哪个好用用哪个。

### 💰 费用完全透明
- ❌ 没有月付/年付订阅陷阱
- ✅ 完全基于源头厂商的 Token 计费
- ✅ 用多少买多少
- ✅ 积极推广免费模型

### 🖥️ 全平台覆盖
支持 **macOS、Windows、Linux、Android、iOS**，一个账号，全端同步，AI 随时在你身边。

### 🔄 灵活切换助手
不同场景用不同 AI。写作、编程、翻译、分析 —— 随时切换最适合的助手。
